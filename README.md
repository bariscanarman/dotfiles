🛠️ My Advanced Linux Development Environment

This repository contains my personal configuration files (dotfiles) for Arch Linux, specifically tuned for my hardware and coding style for Low-Level Development, Embedded Systems, and Edge AI workflows.

   "Minimalist setup for maximum focus."

🖥️ System Overview

> **OS:** Arch Linux (The ultimate playground for kernel-level exploration)
> **WM:** Hyprland (Dynamic tiling Wayland compositor)
> **Terminal:** Kitty (GPU-accelerated for maximum performance)
> **Shell:** Zsh / Bash with custom configurations
> **Bar:** Waybar (Minimalist and informative, not using it rn)
> **App Launcher:** Rofi (Wayland-fork)

📁 Key Configurations

* **hypr/**: Smooth animations, custom keybindings for a seamless multi-monitor workflow.
* **kitty/**: Optimized font rendering and color schemes for long coding sessions.
* **waybar/**: Real-time system monitoring — essential for debugging performance-heavy Edge AI models.
* **rofi/**: Custom themes for quick application access.

  ### 🚀 Why This Setup?

As an Edge AI & Embedded Engineer, I don't just care about looks — I care about every single clock cycle. This environment is my cockpit. (Side note: This config is for my laptop, **Mihawk**)

* **Zero Bloat:** I use Arch and Hyprland to keep the background noise at zero.
* **Performance First:** Every bit of RAM saved is a bit more I can give to CUDA kernels and heavy C++ builds.
* **Workflow:** When I'm deep-diving into Linux Kernel architecture or optimizing YOLO models, I need a system that stays out of my way but gives me all the data (CPU/GPU temps, memory) at a glance.
